# Анализ данных для агентства недвижимости

## Цель проекта: 
Определить самые привлекательные сегменты недвижимости Санкт-Петербурга и Ленинградской области, выявить периоды повышенной активности покупателей и продавцов, а также определить наиболее активные населённые пункты и виды продаваемой недвижимости. [Ссылка на проект]()

## Общий вывод: 

Проведён анализ рынка недвижимости с выявлением ключевых сегментов (двухкомнатных квартир с одним балконом на 4-5 этажах), определены сезонные пики активности продаж и публикаций объявлений. Построен дашборд с визуализацией ключевых показателей.
