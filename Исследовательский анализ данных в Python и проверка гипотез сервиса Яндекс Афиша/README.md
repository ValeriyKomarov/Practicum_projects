# Исследовательский анализ данных в Python и проверка гипотез сервиса Яндекс Афиша

## Цель проекта: 
Проведение исследовательского анализа данных в Python, чтобы выявить инсайты об изменении пользовательских предпочтений и популярности событий осенью 2024 года, а также проверка гипотезы о разнице в поведении пользователей с мобильными и стационарными устройствами.
